What kind of files can I open on Kantan Manga?

Kantan Manga only supports ZIP, RAR, CBZ and CBR files

What CBZ / CBR mean?

CBZ is just a fancy name for a ZIP of images.
CBR is just a fancy name for a RAR of images.

Where can I get CBZ / CBR files?

Because of the nature of the file format, it's not easy to find legal sources. A good way to get some mangas to start reading is to download ブ ラ ッ ク ジ ャ ッ ク に よ ろ し く from here and convert the PDFs to CBZs (remember, a CBZ is a zip file of images)

Can I import EPWING dictionaries?

Yes but not directly. EPWING is a dead undocumented format, trying to deal with it directly is time-consuming and very prone to errors. Instead, Kantan Manga supports dictionaries in the format used by the pop-up dictionary browser extension Yomichan. You should be able to import any Yomichan dictionary into Kantan Manga, this includes EPWINGs converted to Yomichan format. Please note that terms tags are not yet supported, but will be soon.

Where can I get dictionary files?

A list of free dictionaries can be found at the Yomichan documentation . Alternatively you may check this video of Matt vs. Japan where he shows some more dictionaries.
